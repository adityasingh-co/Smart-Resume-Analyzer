
# 🚀 Smart Resume Analyzer

An intelligent web-based application that analyzes resumes using NLP and provides insights, scores, and personalized recommendations to improve employability.

---

## 📌 📖 Project Description

The **Smart Resume Analyzer** is a Streamlit-based web application that allows users to upload their resumes in PDF format and receive instant analysis.

The system extracts key information such as:
- Name
- Email
- Skills
- Education
- Experience

Using NLP techniques, the application:
- Predicts the candidate’s domain (Data Science, Web Development, etc.)
- Identifies missing skills
- Suggests certifications and courses
- Provides YouTube video recommendations
- Generates a resume score based on completeness

An **Admin Dashboard** is also included for:
- Viewing user analytics
- Managing data
- Visualizing insights using graphs

## 🔄 Workflow

1. Upload Resume (PDF)
2. Extract text using PyResParser
3. Apply NLP for structured data extraction
4. Match skills with domain
5. Recommend skills & resources
6. Generate resume score
7. Store data in database
8. Display analytics in dashboard
